Demo Banking Transfer – Database Transaction

Đây là một ứng dụng web đơn giản mô phỏng hệ thống chuyển tiền giữa hai tài khoản ngân hàng nhằm minh họa khái niệm Transaction trong Hệ quản trị Cơ sở dữ liệu.
Hệ thống đảm bảo rằng quá trình chuyển tiền được thực hiện an toàn và nhất quán, tránh tình trạng mất hoặc sai lệch dữ liệu khi xảy ra lỗi.
Mục tiêu của project
Minh họa khái niệm Transaction trong Database
Mô phỏng chuyển tiền giữa hai tài khoản
Thực hiện các thao tác:
 START TRANSACTION
 COMMIT
 ROLLBACK
Đảm bảo tính toàn vẹn dữ liệu theo nguyên tắc ACID
Công nghệ sử dụng
Frontend:
 HTML
 CSS
 JavaScript
Backend:
 Node.js
Database:
 MySQL

 Docker
 Docker Compose
